Detailed Description

Wrapper for UDP socket descriptor.

UDP (User Datagram Protocol) is a lightweight, unreliable, datagram-oriented, connectionless protocol. It can be used when reliability isn't important.

Remember that UDP is not connection-oriented, association functions provided by library just allow you to omit destination address in every write call.

Current restrictions

Current API doesn't supports IP-version agnostic UDP interface. You have to specify IP version in constructor or open() call.

Examples:
echo_client.cpp, and echo_server.cpp.

void libwire::udp::socket::associate ( address  destination,
uint16_t  port 
)
noexcept

Associate remote endpoint with UDP socket.

This allows you to omit destination argument from write call (allows but not requires, you can still pass destination argument and it will override association).

Also you will receive datagrams only from specified source.

This function will replace previous destination information if any.

Note
This function never fail because there is no way to test whether remote endpoint is valid before trying to use it in read or write operation.
Examples:
echo_client.cpp.
