Detailed Description

template<typename Flag, typename Storage = uint32_t>
class libwire::flags< Flag, Storage >

Type-safe wrapper for OR-ed flags.

This class satisfies requirements of BitmaskType concept.

Template Parameters
FlagsUsually enum but also can be any type which have valid bitwise operators that return Storage.
StorageType to use for OR-ed flags store. Following requirement should be met: sizeof(Flag) <= sizeof(Storage).
